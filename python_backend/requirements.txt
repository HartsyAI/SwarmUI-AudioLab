# Do NOT use regular pip install. Follow these steps:

# For Windows (SwarmUI with ComfyUI):
# 1. Navigate to: SwarmUI\dlbackend\comfy\python_embeded\
# 2. Run: python.exe -m pip install -r requirements.txt

# For Linux/Mac:
# 1. Navigate to your SwarmUI directory
# 2. Use the same Python that SwarmUI uses: ./dlbackend/ComfyUI/venv/bin/python -m pip install -r requirements.txt

# Manual installation steps for SwarmUI:
# 
# 1. First, ensure SwarmUI is working properly
# 2. Locate SwarmUI's Python environment:
#    - Windows: SwarmUI\dlbackend\comfy\python_embeded\python.exe
#    - Linux/Mac: SwarmUI/dlbackend/ComfyUI/venv/bin/python
# 
# 3. Install core dependencies:
#    Windows: dlbackend\comfy\python_embeded\python.exe -m pip install fastapi uvicorn numpy scipy
#    Linux/Mac: ./dlbackend/ComfyUI/venv/bin/python -m pip install fastapi uvicorn numpy scipy
# 
# 4. Install STT library (choose one):
#    For RealtimeSTT: python -m pip install RealtimeSTT
#    For basic STT: python -m pip install speech-recognition
# 
# 5. Install TTS library (choose one):
#    For ChatterboxTTS: python -m pip install chatterbox-tts
#    For gTTS: python -m pip install gTTS
# 
# 6. Test installation by running the voice server manually:
#    python voice_server.py --port 7831
# 
# Troubleshooting:
# - If RealtimeSTT fails to install, use speech-recognition instead
# - If ChatterboxTTS fails to install, use gTTS instead
# - For CUDA support, install PyTorch with CUDA first
# - Check SwarmUI logs for specific error messages
